## ğŸ¯ About The Project

**EventHub-AI** is a modern, AI-powered event management system designed to simplify the process of creating, managing, and hosting events. Whether you're organizing a college fest, corporate conference, or cultural event, EventHub-AI provides all the tools you need.

### ğŸ¤” Problem Statement

Event management involves multiple complex tasks:
- Creating detailed schedules manually
- Writing speeches for different occasions
- Managing participant registrations
- Sending timely reminders
- Engaging participants during live events

### ğŸ’¡ Our Solution

EventHub-AI automates these tasks using AI-powered features:
- **AI Schedule Generator** - Creates professional event schedules instantly
- **AI Speech Writer** - Generates customized speeches for any occasion
- **Smart Chatbot** - Answers participant queries 24/7
- **Live Event AI** - Enables real-time interaction during events
- **Automated Reminders** - SMS notifications for participants
